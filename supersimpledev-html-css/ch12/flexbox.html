Main difference from grid is that grid defines column widths in the parent element
and children are automatically inserted into those columns in order; in flex the
parent only says it's flex, and each child defines its own width so more flexible

For example, the youtube header has less buttons available if you aren't signed in,
this is a good place to use flex since we can't define a set number of columns
